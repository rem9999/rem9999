레무링봇 사용자 개인정보 보호 정책 및 처리 방침

레무링봇은 사용자로부터 수집한 데이터를 안전하게 보관중입니다.

수집하는 개인정보

레무링 봇을 이용하기 위한 최소한의 개인정보를 수집합니다. 레무링 봇이 수집하는 개인정보는 아래와 같습니다.
> - Discord 유저 ID | Discord 유저 이름 | Discord 서버 ID | 메시지 콘텐츠 | 디스코드 계정 생성일
레무링 봇은 이용자의 동의 없이 위 개인정보를 수집하지 않습니다.

2. 수집한 개인정보의 이용

레무링 봇은 수집한 개인정보를 아래와 같이 사용합니다.
- 레무링봇을 이용하는 사용자를 구분하기 위해 Discord 유저 ID 및 Discord 서버 ID를 사용합니다.

개인정보가 사용되는 명령어
!가입 - 레무링 봇을 이용하기 위해 필수적으로 사용해야하는 명령어로, Discord 유저 ID, Discord 유저 이름을 데이터베이스에 저장합니다.
!문의 - 레무링봇의 PR 기능입니다. 사용 후 PR 내용 전송 후 Discord 유저 ID 와 Discord 서버 ID가 메시지로 전송되어 기록됩니다.
!배너 - Discord Api를 이용하여 사용자의 프로필 배너를 가져옵니다. 이 데이터는 따로 수집되진 않지만 사용시 다른 사용자에게 프로필 배너 노출됩니다.
!배워 - 레무링 봇을 학습시키는 명령어로 Discord 유저 이름, Discord 유저 ID, 메시지 콘텐츠가 DB에 저장되어 메시지 레무링봇이 사용합니다. 메시지 데이터를 파기하려면 !잊어 명령어를 사용하세요.
!서버설정 - 이모지 확대 및 NSFW 설정을 할 수 있는 명령어 입니다. Discord 서버 ID가 DB에 저장됩니다.
!정보 - Discord API 에서 제공하는 정보를 기반으로 사용자의 정보를 표시합니다. 데이터는 기록되지 않습니다.
!조회 - Discord API 가 제공하는 정보를 기반으로 사용자의 정보를 불러오는 기능입니다. 데이터는 기록되지 않습니다.
!프사 - 사용자의 프로필 사진을 불러옵니다. 데이터는 기록되지 않습니다.

3. 개인정보의 파기

레무링 봇은 사용자가 탈퇴하거나 삭제를 요청하기 전까지 데이터를 안전하게 보관하고 있습니다.
개인정보는 유저가 직접 파기할 수 있습니다.

4. 개인정보 보호

레무링 봇은 이용자의 개인정보를 안전하게 관리하기위해 최선을 다하며, Discord 봇 정책에 따라 개인정보를 보호하고있습니다.
